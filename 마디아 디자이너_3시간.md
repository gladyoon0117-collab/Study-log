# 초기 화면
- community : 피그마 오픈 소스. icon이나 ui/ux 무료로 참고 가능하므로 학습용으로 적절.
- figjam board : 피그마에서 제공하는 또다른 플랫폼. 간단한 ppt 같은 문서작업이나 기록, 발표 자료 만드는 목적으로 쓰임. 회의할 때도 많이 쓰임. 템플릿도 있음. board.
- import : design file 불러오는 기능. 파일 열 게 피그마에 저장돼서 열리는 방식. 파일이 크면 클수록 여는 데 오래 걸림. (폴더 자체에서 drag&drop해도 가능)

# 화면 구성
- 그리는 목적의 tool, 좌측 layer 창, 우측 effect 창으로 구
- file -> save as figma(피그마로 파일 저장), save to version history(지금 버전에서 히스토리 저장하고 싶을 때), show version history(그 저장한 버전 히스토리가 목록으로 보여짐, 피그마 자체적으로도 시간마다 자동저장한 기록도 볼 수 있음)
- page : 모바일 디자인/웹 디자인 이렇게 분리할 때 씀. 무료는 3개가 최대. cover도 만들 수 있음. home에서 한눈에 썸네일로 알아볼 수 있게. cover page에 이미지 넣고 -> 우클릭 -> set as thumnail

# 기초 툴 기능
- background color : 우측 page - 컬러 설정
- frame : 배경판. 모바일/웹 사이즈에 맞게 우측 frame에서 기기 or width, height 값 설정 가능. 어디 디바이스에 맞출지만 고민하면 됨.
- pc 디자인 : 1920x1080
- shift : 비율 유지. 정사각형으로만, 원래 그 비율대로...
- radius : 도형의 round 값 or 우측 패널 도형 -> 더보기(...) -> edit object -> corner radius 설정
- line : 펜툴 기반으로 되어있기 때문에, stroke -> position을 center/inside/outside 선택 -> weight 두께 선택 -> start point와 end point 화살표... 어떤 모양으로 할 건지 선택 가능
- arrow : 화살표가 있는 line. 기획서에서 많이 쓰임.
- 삼각형 : 시각적 그리드를 맞추기 위해 정사각형 프레임 잡혀있음. 싫으면 우클릭 -> outline stroke. 꼭짓점 늘리고 싶으면 count(못찾음) 개수 바꿈. 별도 마찬가지. 딱지나 뱃지 만들 때 유용.
- 이미지 삽입 : 도형 tool 하단 image/video or 이미지 넣을 도형 그린 뒤 fill을 통해 바꿀 수 있음(원하는 size로 한번에 넣고 싶을 때)
- 펜툴 : shift 누르고 그리면 직각/직선 그리는 데 유용. 숙련도 필요한 기능. 펜툴로 그린 것도 corner radius로 round 값 줄 수 있음.
- 단축키 c/v : 화면 움직이는 거(c) 스페이스 바 누른 상태에서도 일시적으로 됨.
- 단축키 z : 확대 모드 (클릭 동안)
- 단축키 z+option : 축소 모드 (클릭 동안)
- 단축키 z+드래그 : 드래그한 영역으로 확대됨. 원하는 위치 바로 확대 가능
- 단축키 command+[+] : 확대
- 단축키 command+[-] : 축소
- 말풍선 : comment. 코멘트 달고 싶은 영역 정확히 드래그해서 코멘트 달 수도 있음.
- 우클릭+cursor chat or 단축키 / : say something. 실시간 채팅 기능. 가만히 있으면 서서히 사라짐. 또는 아무데나 클릭하면 닫힘.
- 우클릭+show/hide UI or 단축키 command+\ or 단축키 command+[.] : 양옆 패널 없이 화면을 좀 더 크게 보고 싶을 때. 다시 되돌릴 때.
- 단축키 command+shift+/(못찾음) : 좌측 레이어 패널만 숨겨짐.

# matching layer
- select matching layers : 단축키 option+command+A. 동시다발적으로 버튼, 폰트, 도형(같은 것)의 서식 등을 수정하는 기능. 두 개 이상도 가능. 그러나 레이어 이름이 같아야 함.
- multi-edit text : 텍스트 두 개 이상 잡은 뒤 단축키 enter 누르고 굵기 같은 거 바꾸면 내용, 굵기까지 동일하게 쌍둥이처럼 바뀜.
- components, assets도 비슷한 기능

# use as mask
- 이미지 뒤로(back) 도형 넣은 뒤, 우측 패널 use as mask 쓰면 도형 안에 이미지 들어감.
- 도형 안에 이미지 위치 조정하고 싶으면 command 누른 상태에서 이미지 조정
