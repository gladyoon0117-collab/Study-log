# 01_Interface_Basic

## 1. Home
- **Community** : 피그마 오픈 소스. 아이콘이나 UI/UX 무료 참고 가능 (학습용 적합).
- **FigJam Board** : board. 협업용 화이트보드 플랫폼. 문서 작업, 기록, 발표 자료 및 회의용 템플릿 제공.
- **Import** : 외부 디자인 파일을 불러오는 기능. 파일 크기에 따라 로딩 시간이 소요됨 (Drag & Drop 가능).

---

## 2. Workspace
- **구조**: 그리기 도구(상단), Layer 창(좌측), Effect/Properties 창(우측).
- **File 메뉴**:
  - `Save as figma`: 로컬 파일 저장.
  - `Save to version history`: 현재 시점의 버전 기록 저장.
  - `Show version history`: 저장된 히스토리 목록 확인 (자동 저장 기록 포함).
- **Page**: 디자인 단위 분리 (웹/모바일 등). 무료 플랜은 최대 3개까지 생성 가능.
- **Cover**: 페이지 우클릭 후 `Set as thumbnail`을 통해 홈 화면 썸네일 설정 가능.

---

## 3. 기초 툴 및 기능
- **Background Color**: 우측 Page 패널에서 설정.
- **Frame**: 배경판(배경지). 우측 패널에서 디바이스별 사이즈 선택 가능 (PC 표준: 1920x1080).
- **Shape 활용**:
  - `Shift` 누르고 드래그: 정비율 유지.
  - **Radius**: 도형의 라운드 값 조절. (`Edit object` -> `Corner radius`)
  - **Line/Arrow**: 펜툴 기반. 두께(`Weight`), 위치(`Position`), 화살표 모양 등 설정 가능.
  - **Polygon/Star**: `Count` 설정을 통해 꼭짓점 개수 조절 가능.
- **Image**: 도형 생성 후 `Fill`을 통해 삽입하거나 단축키로 이미지/비디오 삽입.
- **Pen Tool**: `Shift`를 활용해 직각/직선 그리기 용이.

---

## 4. 단축키 및 화면 제어
| 기능 | 단축키 | 비고 |
| :--- | :--- | :--- |
| **화면 이동** | `Space` + 드래그 | `H`와 동일 |
| **확대 모드** | `Z` + 클릭 | 일시적 확대 |
| **축소 모드** | `Z` + `Option` + 클릭 | 일시적 축소 |
| **영역 확대** | `Z` + 드래그 | 원하는 영역 집중 확대 |
| **확대/축소** | `Cmd` + `+` / `-` | |
| **코멘트(댓글)** | `C` | 원하는 영역 지정 가능 |
| **커서 채팅** | `/` | 실시간 소통 (작성 후 사라짐) |
| **UI 숨기기** | `Cmd` + `\` 또는 `.` | 패널 없이 화면 보기 |

---

## 5. 고급 레이어 편집 (Matching & Mask)
- **Matching Layer**: `Option` + `Cmd` + `A`. 이름이 같은 레이어를 동시 선택하여 서식 일괄 수정.
- **Multi-edit Text**: 여러 텍스트 선택 후 `Enter`를 통해 내용 및 속성 동시 수정.
- **Mask**: 이미지 뒤에 도형 배치 후 `Use as mask` 적용.
  - 레이어에서 개별 선택하여 위치 및 라운드 조절 가능.
- **Group/Frame**: `Cmd` + `G` (그룹화) 또는 `Frame selection` (프레임 안에 프레임).

---

## 6. 불린 연산 (Boolean Groups)
- **Union**: 도형 합치기.
- **Subtract**: 상위 레이어 도형만큼 구멍 뚫기.
- **Intersect**: 교집합만 남기기.
- **Exclude**: 교집합만 뚫기.
- *Tip: `Cmd`를 누른 채 클릭하면 합쳐진 상태에서도 개별 도형 위치/라운드 수정 가능.*

---

## 7. 정렬 (Align & Spacing)
- **Alignment**: 상단/하단/좌우/중앙 정렬.
- **Tidy Up**: 요소들을 바둑판 형태로 자동 정렬.
- **Distribute Spacing**: 수직(`Vertical`) 또는 수평(`Horizontal`) 간격을 동일하게 배분.
