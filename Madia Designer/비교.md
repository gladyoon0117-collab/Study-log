# 01_Interface_Basic

## 1. Home
- **Community**: 피그마 오픈 소스. 아이콘이나 UI/UX 무료 참고 가능 (학습용 적합).
- **FigJam Board**: board. 협업용 화이트보드 플랫폼. 문서 작업, 기록, 발표 자료 및 회의용 템플릿 제공.
- **Import**: 외부 디자인 파일을 불러오는 기능. 파일 크기에 따라 로딩 시간이 소요됨 (Drag & Drop 가능).



## 2. Workspace
- **구조**: 그리기 도구(상단), Layer 창(좌측), Effect/Properties 창(우측).
- **File 메뉴**:
  - `Save as figma`: 로컬 파일 저장.
  - `Save to version history`: 현재 시점의 버전 기록 저장.
  - `Show version history`: 저장된 히스토리 목록 확인 (자동 저장 기록 포함).
- **Page**: 디자인 단위 분리 (웹/모바일 등). 무료 플랜은 최대 3개까지 생성 가능.
- **Cover**: 페이지 우클릭 후 `Set as thumbnail`을 통해 홈 화면 썸네일 설정 가능.



## 3. 기초 툴 및 기능
- **Background Color**: 우측 Page 패널에서 설정.
- **Frame**: 배경판(배경지). 우측 패널에서 디바이스별 사이즈 선택 가능 (PC 표준: 1920x1080).
- **Shape 활용**:
  - `Shift` 누르고 드래그: 정비율 유지.
  - **Radius**: 도형의 라운드 값 조절. (`Edit object` -> `Corner radius`)
  - **Line/Arrow**: 펜툴 기반. 두께(`Weight`), 위치(`Position`), 화살표 모양 등 설정 가능.
  - **Polygon/Star**: `Count` 설정을 통해 꼭짓점 개수 조절 가능.
- **Image**: 도형 생성 후 `Fill`을 통해 삽입하거나 단축키로 이미지/비디오 삽입.
- **Pen Tool**: `Shift`를 활용해 직각/직선 그리기 용이.

## 4. 단축키 및 화면 제어
| 기능 | 단축키 | 비고 |
| :--- | :--- | :--- |
| **화면 이동** | `Space` + 드래그 | `H`와 동일 |
| **확대 모드** | `Z` + 클릭 | 일시적 확대 |
| **축소 모드** | `Z` + `Option` + 클릭 | 일시적 축소 |
| **영역 확대** | `Z` + 드래그 | 원하는 영역 집중 확대 |
| **확대/축소** | `Cmd` + `+` / `-` | |
| **코멘트(댓글)** | `C` | 원하는 영역 지정 가능 |
| **커서 채팅** | `/` | 실시간 소통 (작성 후 사라짐) |
| **UI 숨기기** | `Cmd` + `\` 또는 `.` | 패널 없이 화면 보기 |

## 5. Matching Layer
* **Select Matching Layers** (`Opt` + `Cmd` + `A`): 레이어 이름이 동일한 여러 요소(버튼, 폰트, 도형 등)를 동시 선택하여 서식을 일괄 수정하는 기능.
* **Multi-edit Text**: 텍스트 레이어 여러 개 선택 후 `Enter` 입력 -> 내용, 굵기 등을 쌍둥이처럼 동시에 변경 가능.
* **기타**: Components, Assets도 이와 유사한 효율화 기능을 가짐.

---

## 6. Use as Mask
* **기본 원리**: 이미지 뒤로 도형을 배치한 뒤 `Use as Mask` 실행 -> 도형 안에 이미지가 들어가는 방식.
* **세부 조정**:
    * **이미지 위치**: 레이어 패널에서 `IMG` 선택 후 이동 및 조절.
    * **도형 라운드**: 레이어 패널에서 `Rectangle` 선택 후 조정.
* **그룹 관리**: 개별 요소가 따로 놀지 않도록 전체 선택 후 `Cmd` + `G` 또는 `Frame Selection`으로 묶어서 관리 권장.

---

## 7. Union Selection
* **Union**: 두 개 이상의 도형을 하나로 합치는 기능.
* **Subtract**: 상위 레이어(Front) 도형의 교집합 영역만큼 아래가 뚫림.
* **Intersect**: 교집합 영역만 남김.
* **Exclude**: 교집합 영역만 뚫음.
* **상세 팁**:
    * `Cmd` 누르고 클릭 시 합쳐진 상태에서도 개별 도형의 위치나 라운드 값 수정 가능.
    * `Ungroup` 시 다시 개별 도형으로 분리됨.
    * 이미지와 도형 혼합 시 더 상위 레이어 기준으로 덮임.

---

## 8. Align
* **Alignment**: 우측 패널 상단 위치. 요소 전체를 좌/우/중앙/상/하로 정렬.
* **Tidy Up**: 바둑판 형태 자동 정렬. 수직/수평 간격을 한 번에 조정해 줌.
* **Distribute Spacing**:
    * `Vertical`: 수직 간격을 동일하게 배분.
    * `Horizontal`: 수평 간격을 동일하게 배분.

---

## 9. Size & Angle
* **Position (X, Y)**: 도형의 좌표(위치) 값.
* **Layout (W, H)**: 가로 및 세로 수치.
* **비율 유지(클립 아이콘)**: 활성화 시 가로/세로 비율을 고정하여 크기 조정 가능 (예: H를 300에서 1200으로 변경 시 W도 비율에 맞춰 자동 증가).
* **Rotation**: 각도 조절. 커서로 직접 회전 가능하나, 정확한 설계를 위해 우측 패널 수치 입력을 권장함.
- **Group/Frame**: `Cmd` + `G` (그룹화) 또는 `Frame selection` (프레임 안에 프레임).
